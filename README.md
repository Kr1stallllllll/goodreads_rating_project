# Проект: Прогнозирование рейтинга книг на Goodreads

## Описание проекта
Этот проект направлен на разработку модели машинного обучения для предсказания среднего рейтинга книг на основе метаданных, доступных на платформе Goodreads. Задача относится к области регрессии и помогает выявить ключевые факторы, влияющие на успех книги у читателей.

## Цель проекта
Создать систему, которая принимает на вход параметры книги (например, количество страниц, язык, количество отзывов) и предсказывает её вероятную оценку (от 0 до 5 звезд).

## Данные
- **Источник:** Датасет Goodreads Books (Kaggle).
- **Характеристики:**
    - `num_pages` (количество страниц)
    - `language_code` (язык издания)
    - `ratings_count` (количество оценок)
    - `text_reviews_count` (количество текстовых отзывов)
- **Целевая переменная:** `average_rating` (средний рейтинг).

## Инструменты
- Язык: Python 3.10+
- Библиотеки: Pandas, NumPy, Scikit-learn, Gradio.
- Управление версиями: Git.
